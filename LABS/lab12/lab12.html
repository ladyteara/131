The purpose of this lab is to practice working with singly linked structures. You need to copy the module node.py into the folder that contains your solution for this lab. 

Write the code for a class called SinglyLinkedStructure that represents a singly linked structure.

The class contains:         

  • A constructor with two instance variables head and size, where
    • head is a link to the first node in the singly linked structure. It should be set to None when the linked structure is empty.
    • size is the number of nodes currently in the list.
    The constructor accepts one parameter called seq with default value None, which is expected to receive a Python sequence. The constructor uses seq to initialize the singly linked structure. For example, if the Python list [1,2,3] is plugged in for seq, the constructor will create a singly linked structure object with values 1, 2, 3 where the first node contains the value 1, the second node contains the value 2, and so on.
  •  An instance method append(value) that appends a new node with data set to value to the end of the singly linked structure. Notice that your constructor can use this method to append the values in seq to the singly linked structure.
  • An instance method __len__ that returns the number of nodes in the singly linked structure.
  • An str method that returns a string representation of the singly linked structure consisting of the concatenation of the values in the different nodes.
  • Use the following main() function to test your code: 
    def main():
    l1 = SinglyLinkedStructure()
    print('Empty singly linked structure')
    print(l1)
    print(len(l1))
    print()
    
    l1.append('Hello,')
    l1.append('how')
    l1.append('are')
    l1.append('you?')
    print(l1)
    print(len(l1))
    print()
    
    seq = list(range(1,6))
    l2 = SinglyLinkedStructure(seq)
    print(l2)
    print(len(l2))
    print()

    t = tuple([x*10 for x in seq])
    print(t)
    print()

    l2 = SinglyLinkedStructure(t)
    print(l2)
    print(len(l2))
    print()

    l2 = SinglyLinkedStructure("abcxyz")
    print(l2)
    print(len(l2))
    
main()
